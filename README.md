# Music Player

A MILI UI demo featuring a rich music player.

## Features

### Playlists

- Rename
- Delete
- Empty/Load from folder
- Reorder
- Reset/Upload/Auto-generate cover

### Musics

- Rename
- Add/Delete
- Reorder
- Change playlist
- Score-based Search

### Player

- Volume/Mute
- Play/Pause
- Next/Previous/Auto-next
- Music loop
- Playlist loop
- Playlist shuffle
